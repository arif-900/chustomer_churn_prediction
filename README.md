# Customer Churn Prediction — Dashboard

This repository contains a Streamlit dashboard for exploring and modeling the `Customer_churn_prediction.csv` dataset.

Run locally:

```powershell
pip install -r requirements.txt
streamlit run app.py
```

What the app provides:
- Dataset preview
- Target distribution and categorical breakdowns
- Numeric histograms and correlation heatmap
- Quick RandomForest training with selectable features and basic metrics
